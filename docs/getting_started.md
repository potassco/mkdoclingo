# Getting started

## Installation

=== "Pip"

```console
pip install mkdoclingo
```

=== "Development mode"

```console
git clone https://github.com/potassco/mkdoclingo.git/
cd mkdoclingo
pip install -e .[all]
```

!!! warning
    Use only for development purposes

## Usage

### Command line interface

Details about the command line usage can be found with:

```console
mkdoclingo -h
```
