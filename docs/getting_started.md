# Getting started

## Installation

=== "Pip"

```console
pip install fillname
```

=== "Development mode"

```console
git clone https://github.com/potassco/fillname.git/
cd fillname
pip install -e .[all]
```

!!! warning
    Use only for development purposes

## Usage

### Command line interface

Details about the command line usage can be found with:

```console
fillname -h
```
